Description

I will teach you how to set up an Ubuntu Server. I will show you how I configure updates, time, and SSH. This is a good starting point for your first home lab. From there, you can start building higher.

Grabbing Files

Let's first start with grabbing the ISO and Ventoy so we can boot our live environment. Here you can download the Ubuntu Server ISO. Here you can download Ventoy for making a bootable USB drive. Store these files in your download directory.

Creating a Bootable USB Drive with Ventoy

First, in Windows, we need to right-click on the file and select "Extract All." After that, run Ventoy2Disk as admin.

In Linux, you will need to install tar and then run the following command: 

tar -xvf ventoy*

After that, launch VentoyGUI with this command: 

./VentoyGUI.x86_64

After you have Ventoy open, select your USB drive and click on "Install". This will wipe everything from your USB drive, so make sure to back up any data that you have on your USB drive.

Now you can copy the ISO file to your Ventoy USB drive. After that, you can reboot your PC and press the F key to boot from the USB drive. I apologize for not being able to specify which F key to press, as it varies depending on your specific motherboard. I recommend checking your PC's manual or motherboard manual or manufacturer's website for information on how to access the boot menu and select a different boot device.

Running the Installer and Installing Ubuntu

After booting the Live ISO, run the installer and select what suits you, such as language, timezone, username, and password.

I recommend selecting the "Minimal Install" option and installing third-party drivers.

I always set a static IP address on my server. This will make it easier to access your services later.

Change the IP address to be the same as your subnet's. This is an example.

Then, apply your changes and continue with the installer.

I recommend that beginners select the default options, while more advanced users may want to consider using RAID or ZFS.

If everything looks correct, you can press Done and continue.

Now, select the username and password.

Now, continue with the installation process.

I recommend installing SSH, it will make it easier to remotely control your server.

Now that the installation process is complete, you can reboot your server and safely remove or unplug your USB drive.

Congratulations! You have successfully installed Ubuntu Server!

Final Setup: Time Settings, Automatic Updates & Reboots

Your Ubuntu Server is now up to date. Let's take care of a few final settings.

Set the Correct Timezone

Run the following command and follow the menu to select your region: 

sudo dpkg-reconfigure tzdata
Enable Automatic Security Updates

Ubuntu can automatically install security updates. Use this command: 

sudo apt install unattended-upgrades

Make sure it's active: 

sudo dpkg-reconfigure --priority=low unattended-upgrades
Prevent Automatic Reboots

Ubuntu wonâ€™t restart your server automatically unless you enable that. To be sure it's off, check this file: 

sudo nano /etc/apt/apt.conf.d/50unattended-upgrades

Look for this line inside the file: 

Unattended-Upgrade::Automatic-Reboot "false";

If it says "true", change it to "false" and save. and dont forget to uncomment the line by removing the // at the start of the line. pres control + o to save and control + x to exit.
