Description

In this article, I'll teach you how to set up an Ubuntu Server. You will learn how to configure updates, set up correct system time and SSH. This is an ideal starting point for your first home lab. From here, you can keep building further.

Grabbing Files

Let's start by downloading the ISO and Ventoy, so we can boot into a live environment.
Download the Ubuntu Server ISO.
Download Ventoy to create a bootable USB drive.
Save both files in your Downloads folder.

Creating a Bootable USB Drive with Ventoy

In Windows:

  1. Right-click on the Ventoy file and choose "Extract All."
  2. Run Ventoy2Disk as administrator.

In Linux:

  1. Install tar if needed and extract Ventoy:
tar -xvf ventoy*

Then start the GUI with:

./VentoyGUI.x86_64

Once Ventoy is running, select your USB drive and click "Install". Warning: This will erase all data on the USB drive! Be sure to back up any important data first.

After installing Ventoy on the USB drive, copy the Ubuntu ISO to the USB. Then reboot your PC and use the correct F key to open the boot menu. Since this key differs per motherboard, check your PC's manual or the manufacturer's website.

Running the Installer and Installing Ubuntu

Boot from the live ISO, start the installer, and select your preferred language, timezone, username, and password.

I recommend selecting the "Minimal Install" option and installing third-party drivers.

It's wise to set a static IP address for your server; this makes accessing your services later a lot easier.

Apply your changes and proceed with the installer.

Beginners: use the default partition settings. Advanced users may want to use RAID or ZFS.

When you're happy with the layout, press "Done" to continue.

Set your username and password.

Proceed with the installation.

I strongly recommend installing SSH; it makes remote server management much easier.

When the installation is finished, reboot your server and remove the USB drive.

Congratulations! You have successfully installed Ubuntu Server!

Now, let's configure Ubuntu Server properly

After installing Ubuntu Server, there's some configuration left: setting up updates, adjusting time settings, disabling auto-reboot after updates, and enabling automatic updates.

Manual Update for Ubuntu Server

First, log in to your server.
Then update Ubuntu with the following command: 

sudo apt update && sudo apt upgrade -y && sudo apt install bash-completion

The bash-completion tool adds handy tab-completion in the shell.

Final Setup: Time Settings, Automatic Updates & Reboots

Your Ubuntu Server is up to date. Let's finalize some key settings.

Set the Correct Timezone

Run this command and select your timezone:

sudo dpkg-reconfigure tzdata

Enable Automatic Security Updates

To enable automatic security updates, run:

sudo apt install unattended-upgrades

Then make sure it is enabled:

sudo dpkg-reconfigure --priority=low unattended-upgrades

Prevent Automatic Reboots

By default, Ubuntu won't restart automatically, but to be sure, check this file:

sudo nano /etc/apt/apt.conf.d/50unattended-upgrades

Inside the file, ensure you see:

Unattended-Upgrade::Automatic-Reboot "false";

If it says "true", change it to "false" and uncomment the line (remove // at the start).
Press Ctrl + O to save, then Ctrl + X to exit.
